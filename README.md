# murmur

Xのモバイルアプリのように、短い言葉を自分だけのタイムラインへ残せるADHD向けジャーナリングPWAです。

## 起動

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。iPhoneのSafariでは共有メニューから「ホーム画面に追加」すると、アプリのように起動できます。

## 主な機能

- 500文字までの短文ジャーナリング
- 気分とタグ（気づき、モヤモヤ、できた、あとで）
- 自分だけの時系列タイムライン
- 検索、絞り込み、大切なメモ、編集、削除、コピー
- LocalStorageによる端末内保存
- オフライン動作とホーム画面への追加

入力内容はサーバーへ送信しません。ブラウザのデータを消去するとメモも消えるため、製品化時にはiCloud等への暗号化バックアップを追加してください。

# murmur

Xのモバイルアプリのように、短い言葉を自分だけのタイムラインへ残せるADHD向けジャーナリングPWAです。

## 起動

```bash
python3 -m http.server 8082
```

ブラウザで `http://localhost:8082` を開いてください。iPhoneのSafariでは共有メニューから「ホーム画面に追加」すると、アプリのように起動できます。

## 主な機能

- 500文字までの短文ジャーナリング
- 気分とタグ（気づき、モヤモヤ、できた、あとで）
- 自分だけの時系列タイムライン
- 検索、絞り込み、大切なメモ、編集、削除、コピー
- 「#あとで」付き投稿から自動で作るToDoリスト（完了状態の保存・未完了絞り込み）
- LocalStorageによる端末内保存
- 選んだ期間のポストをChatGPTに貼り付ける文章へまとめるふりかえり
- オフライン動作とホーム画面への追加

通常のポストとふりかえり用に作成した貼り付け文は端末内に保存されます。ふりかえりはOpenAI APIを使わず、対象期間の本文・日付・気分・タグをChatGPTに貼り付けやすい文章へ整形します（画像は含めません）。コピーした内容をChatGPTに貼り付けた時点で、選んだポストの内容がChatGPTへ送信されます。

ブラウザのデータを消去するとメモも消えるため、製品化時にはiCloud等への暗号化バックアップを追加してください。
